שאלות קשות — תשובות ישרות

מה שהקהל שואל בסוף כל דמו

מה קורה כשה-AI טועה? אתם אחראים על הגשה לרשויות?
תשובה ישרה: ה-AI הוא כלי עזר — האדריכל הרשום תמיד חותם ונושא באחריות מקצועית. הכלי מפחית טעויות אנוש ב-QC (הוא לא שוכח לבדוק ת"י 1918), אבל לא מחליף שיקול דעת מקצועי. כל פלט עובר עיניים אנושיות לפני הגשה. סיכון נמוך כי הוא עושה את העבודה המשעממת, לא את ההחלטות.
כמה עולה? האם ROI אמיתי או שמחירי API יאכלו את החיסכון?
עלויות אמיתיות: Claude API כ-$15/מיליון טוקן. QC מלא של 847 אלמנטים כ-80K טוקן = $1.20. מפרט טכני מלא כ-200K טוקן = $3.00. Veras renders = $3/תמונה (קבוע). חיסכון ממוצע לפרויקט: 15–30 שעות עבודה. בתעריף ₪150/שעה = ₪2,250–₪4,500 לפרויקט. ROI: ×300–×500 על עלות ה-API. חישוב שמרני
נתוני הלקוח עולים ל-cloud — מה עם סודיות?
Anthropic Enterprise: נתונים לא נשמרים לאימון בחוזה enterprise. אפשרות הרצה מקומית דרך AWS Bedrock / Azure AI. Revit MCP רץ locally — הקובץ לא עולה, רק metadata. בפרויקטים רגישים: AWS Bedrock עם VPC private endpoint, אפס יציאה לאינטרנט.
כמה זמן לקח ללמד את הצוות? האם כל עובד צריך להיות מתכנת?
לא צריך קוד. כל הממשק הוא שפה טבעית בעברית. ההטמעה: יום הדרכה ×2 (אדריכלים + עוזרים). קצב אימוץ מהשטח: 80% מהצוות עצמאי אחרי 3 פרויקטים. האתגר האמיתי הוא לא טכני — הוא שינוי הרגלים. אימון: יום אחד לרמה בסיסית, 3 שבועות לשטף מלא.
